📋 The Problem

Why This Project Exists

Solving a real operational challenge at IPB University

😩

The Problem

Panitia MPKMB harus memverifikasi 5.000+ bukti pembayaran (gambar & PDF) secara manual. Setiap file harus dicek satu per satu — apakah penerima GoPay? Apakah tujuan PB MPKMB DRAMAGA?

⏰ Memakan waktu berhari-hari 😵 Rentan human error 📱 Format gambar beragam
🚀

The Solution

Sistem otomatis yang menggunakan Computer Vision dan OCR untuk membaca teks dari 5.000+ bukti pembayaran, decode QR code, dan memfilter menjadi 580 file terkurasi dengan akurasi ~95%.

⚡ Proses otomatis dalam menit 🎯 Akurasi tinggi dengan multi-pass OCR 📄 Support gambar & PDF
✨ Features

What It Can Do

def preprocess_image(image):
    results = []

    # 1. Grayscale
    if len(image.shape) == 3:
        gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    else:
        gray = image.copy()
    results.append(gray)

    # 2. Adaptive Threshold
    adaptive = cv2.adaptiveThreshold(
        gray, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
        cv2.THRESH_BINARY, 11, 2)
    results.append(adaptive)

    # 3. OTSU Threshold
    _, otsu = cv2.threshold(
        gray, 0, 255,
        cv2.THRESH_BINARY + cv2.THRESH_OTSU)
    results.append(otsu)

    # 4. CLAHE Enhancement
    clahe = cv2.createCLAHE(
        clipLimit=2.0, tileGridSize=(8, 8))
    enhanced = clahe.apply(gray)
    results.append(enhanced)

    return results

def extract_text_ocr(reader, image):
    all_texts = []
    image = resize_if_needed(image)
    preprocessed = preprocess_image(image)

    # OCR on grayscale
    try:
        results = reader.readtext(
            preprocessed[0],
            detail=0, paragraph=True)
        all_texts.extend(results)
    except Exception:
        pass

    # Fallback: enhanced contrast
    if len(all_texts) < 3:
        try:
            results = reader.readtext(
                preprocessed[3],
                detail=0, paragraph=True)
            all_texts.extend(results)
        except Exception:
            pass

    return all_texts

def check_gopay_recipient(texts, qr_data):
    combined = " ".join(texts).lower()
    qr_text = " ".join(qr_data).lower()
    all_text = " ".join(
        (combined + " " + qr_text).split())

    # 1. Exact phrase matching
    for kw in GOPAY_RECEIVER_KEYWORDS:
        if kw in all_text:
            return True, kw

    # 2. Context-aware detection
    has_gopay = any(
        g in all_text
        for g in GOPAY_GENERAL)
    if has_gopay:
        is_sender = any(
            s in all_text
            for s in GOPAY_SENDER_KEYWORDS)
        has_recv = any(
            r in all_text
            for r in ["penerima",
                "ditransfer ke", "tujuan"])
        if has_recv and not is_sender:
            return True, "gopay (context)"

    return False, None
